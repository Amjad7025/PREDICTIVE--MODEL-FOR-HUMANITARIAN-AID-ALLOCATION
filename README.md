# PREDICTIVE--MODEL-FOR-HUMANITARIAN-AID-ALLOCATION


## PROJECT OVERVIEW

Predicts regions needing urgent humanitarian support,Uses data like population, disaster impact, and resources,Applies machine learning for priority allocation,Improves fairness and efficiency of aid distribution,Supports faster disaster response

## AIM

Build an automated detection system for AI-generated text

## MODEL DETAILS


* Base Model :


   - GPT-2 (transformer-based language model)

* Tokenizer :
  
   - GPT2TokenizerFast

* Metrics Used :

  - Burstiness:

   Measures variation in sentence length

  - Perplexity:

   Measures how predictable the text is

* LOGIC :
  
   - Heuristic thresholds classify text as AI / Possibly AI / Human.



## PROCESS

* Collect Input
* Tokenize Text
* Calculate Perplexity
* Calculate Burstiness
* Apply Heuristic Logic
* Return Prediction


## RESULT 
AI-generated: highly fluent, low burstiness → classified as Likely AI

Human-written: irregular sentence lengths → classified as Likely Human

IT IS GIVE GOOD AND ACCURATE ANSWERS 


## CONCLUSION

The Predictive Model for Humanitarian Aid Allocation uses data analysis and machine learning to identify regions that need urgent support during crises.
It helps improve the speed, fairness, and efficiency of distributing humanitarian resources to affected populations.
